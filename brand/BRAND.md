## Colours

| Name | Hex | Usage |
|------|-----|-------|
| **Sunburst** (primary) | `#FB8B0D` | Accent, CTAs, highlights |
| **Charcoal** | `#303030` | Primary text, headings |
| **Slate** | `#6C7983` | Secondary text, subheadings |
| **Ivory** | `#EEEBE4` | Backgrounds, light sections |
| **Onyx** | `#000000` | High-contrast text only |
